GoToMenu is a plugin that allows users to press the F2 key on the frontend to open a modal with a dropdown list of available menus, enabling easy navigation.

== Description ==

GoToMenu is a lightweight WordPress plugin that enhances user navigation by enabling the F2 key to open a modal with a dropdown list of available menus on the frontend. The plugin settings allow administrators to toggle this feature on or off.

== Installation ==

1. Upload the `gotomenu` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings under 'Settings' -> 'GoToMenu'.

== Frequently Asked Questions ==

= How do I enable the GoToMenu feature? =
Navigate to the plugin settings under 'Settings' -> 'GoToMenu' and toggle the 'Enable for Frontend' option.
